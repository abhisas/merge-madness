# Merge Madness

## Description
This repository is a starting point for our class project. 
It contains all the foundational code and documentation we will need. 
Please read through this document carefully before making changes.

## Prerequisites
Make sure you have the following before you begin:
* Code Editor (VS Code recommended)
* Terminal or Command Line Interface
* Node.js v18.0.0 or higher
* A good attitude for learning

## Installation
Follow these steps to get the project running locally:
1. Clone this repository to your local machine.
2. Navigate into the project folder using your terminal.
3. Run the setup script.
